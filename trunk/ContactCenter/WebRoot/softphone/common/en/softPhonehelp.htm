Telephone
1、How to use keypad?
A、1. Click the keypad button, and the keypad will be displayed.

B、Key functions
Number keys: For outbound calls or transfers, the key presses will be displayed in the phone number input area..
Backspace: Delete parts of the contents in the phone number input area.
Delete: Delete all of the contents in the phone number input area.
Redial: Redial the phone number that was dialed previously.
Quick dialing: The system enables 7 quick dial keys to be programmed in the configuration file.
Function keys: In the default it's unusable of the function key, see also the chapter of "Customize of Softkey miscellaneous function" of EZactor_Developer_Guide.doc.
Online help: System provides online help.
Settings: This function is mainly used for customization.

C、Contact history
System will record the last inbound/outbound phone number and time, so the agent can redial this phone number in future. The maximum number of records is 20 phone numbers, in descending order by time

D、Address book
this is used for customization.

E、Click on the keypad button again to close the relevant tabs.

4、Transferring a Call
Transfer a call means transferring a call to another agent, which is a usual occurrence in a Call Center. For example, Agent A has answered a call from a customer, but the relevant service requires Agent B’s processing; Agent A could then transfer this call to Agent B.

Note:
When transferring a call, the call must already be connected.
A、One-step transfer
Input the phone number that the call is to be transferred to in the phone number input area, or select the agent that the call is to be transferred to from the web page. Click and the system will transfer the call, while the current agent's call will hang up
B、Multi-step transfer
Input the phone number that you want to transfer in telephone input box, or select agent from floating page directly, then click , current line will be hold , after the orther side accept the call, click again. Transfer successfully and the line of current agent will be hung up.
Note:
The current transfer can be cancelled when the transfer has been confirmed, transfer line has been chosen, and then the call will hang up. At this time the line being transferred is still on hold, and the ‘Hold’ button can be clicked to cancel hold. Both one-step and multi-step transfers can be configured.

5、 Initiating a Conference Call
A conference call means inviting other agents to join a conference during a call. For example, Agent A has answered a call from a customer, but the relevant service requires Agent B's assistance, so Agent A could invite Agent B to join the conference.

Note :
During a conference call, the call must have already been connected .

A、One-Step Conference
Input the third-party phone number in telephone input box, or select agent who need to attend the conference from floating page directly, then click, you can make a three-way talking.

B、Multi-step Conference
Input the third-party phone number in telephone input box directly, or select agent who need to attend the conference from floating page, then click , system will hold current line and dial , after clicking again , you can make a three-way talking.

Note :
The current conference can be cancelled when the call has been confirmed, invited line has been selected, and the call will hang up. At this time, the invited line has been hung up, the main line is still on hold, and the ‘Hold’ button can be clicked to cancel the hold.
System supports multilateral conference calls .

6、Holding a Call
Holding a call allows a call to be held whilst still being connected, and a customer will hear background music instead of the agent’s voice.

Note :
When holding a call, the call must have been already connected.
System supports simultaneous hold of multiple lines.
Hold: select the line that is to be placed on hold, click and the system will hold the current line.
Cancel Hold: select the line that has been placed on hold, click and the system will cancel hold and then resume the call.

7、Transferring a Call to IVR
The IVR process design can be done in Admin Center. An agent can transfer a call to auto voice during a call, which will help complete the dialogue with customers.

Note :
This can only be executed when the call has been connected.
System supports 2 types of IVR transfer: Return to agent; and not return to agent. The differences are.
① • Not return to agent: the call is transferred to IVR directly, which will not be returned to the agent. In regards to the agent, the call service has been finished.
② • Return to agent: the call is transferred to IVR, the agent will hold the call, and after customer and IVR communication is complete, the call will return to the agent.
