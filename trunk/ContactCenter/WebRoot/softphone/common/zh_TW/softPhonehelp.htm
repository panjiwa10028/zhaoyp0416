電話
1、如何使用軟鍵盤
A、點選軟鍵盤按鈕，顯示下圖軟鍵盤頁面。

B、各功能按鍵使用
數字鍵： 點選數位鍵可直接顯示在電話輸入框。
回格鍵Backspace： 清除電話輸入框中的部分內容。電話輸入框中游標定位，點此鍵清除游標前的一個字元。電話輸入框中選中部分或全部，點此鍵清除選中的字元。
清除鍵： 清除電話輸入框中的全部內容。
重撥鍵： 對本次登錄中的最後一次撥打的電話進行重新外撥。使用線路為當前選中線路。
快速撥號鍵： 系統提供在設定檔中設置七個快速撥號鍵，方便座席直接點擊數位鍵進行快速撥號。
功能鍵： 功能鍵默認為不可用，具體設置見 EZactor_Developer_Guide.doc 中關於“ 軟鍵盤協助工具鍵客制化”的說明部分。
Online help： 系統提供的線上說明。
設置： 此功能主要是進行客制化使用的，可由使用者根本自己的需要進行設計。

C、聯絡歷史

系統記錄當前座席最近聯絡的呼入與呼出的電話號碼和時間，目的是方便座席直接選擇已聯絡的電話號碼直接進行再次撥打。
系統最大顯示電話條數為20條，並且是按撥打或接通時間的降冪排序的，也就是說總將最近撥打或接通的電話放在最上面。

D、地址簿
此頁面是由使用者自行定制，可根據需求進行客制化。

E、再次點選軟鍵盤按鈕，將關閉相關頁面。

2、如何外撥電話
1、 在電話輸入框，通過鍵盤或軟鍵盤輸入外撥電話號碼
2、 點選 ，進行撥號，此時撥號按鈕顯示為 ，電話接通時，撥號按鈕將顯示為 。
3、 電話接通後，通話計時開始計時 。
4、 系統並彈出聯絡小結頁面，座席可以在通話或通話結束後做聯絡小結。具體如何使用聯絡資訊，請參看下面的如何在通話中使用聯絡資訊。

3 、如何接進線電話
1、 電話進線後，會在電話輸入框顯示進線的電話號碼 。
2、 點選 ，進行接通電話。
3、 電話接通後，通話計時開始計時 。
4、 系統並彈出聯絡小結頁面，座席可以在通話或通話結束後做聯絡小結。具體如何使用聯絡資訊，請參看下面的如何在通話中使用聯絡資訊。

Note:
業務定位，可以在業務管理中設置相關的進線代號或隨路數據，系統根據進線電話進行業務定位。方便座席在進線響鈴時就能很快瞭解到客戶需要諮詢的業務，以便進行更快捷的服務。

4、如何轉接電話
轉接電話，就是將通話中的電話轉接給另外的座席處進行處理，一般在客服中心運用較多。如：座席A接到客戶電話，但此客戶對應的業務服務需要座席B處理，則座席A可以將此通電話轉給座席B進行處理。

Note:
轉接電話時，必須保持電話為通話中才可以進行此操作。
A、單步轉接
在電話輸入框中輸入需轉接的電話號碼或在浮動頁面中座席狀態查看中選擇轉接的座席，點擊 ，系統將通話的線路直接轉接到相關電話中，當前座席的線路進行掛斷。
B、多步轉接
在電話輸入框中輸入需轉接的電話號碼或在浮動頁面中座席狀態查看中選擇轉接的座席，點擊，系統會對當前使用線路進行Hold，待對方接起後，再次點擊進行轉接確認，系統將通話的線路直接轉接到相關電話中，當前座席的線路進行掛斷。
Note:
在您做確認轉接時，可以取消當前轉接動作，點選轉接的線路，進行電話掛斷。此時轉接的線路為掛斷，被轉接的線路仍Hold，您如需取消Hold，可以再點擊Hold按鈕進行取消Hold。使用單步轉接和多步轉接，可進行配置。

5、如何會議電話
會議電話，就是在通話中時，可以將其他座席邀請進來，一起做電話會議。如：座席A接到客戶電話，但此客戶對應的業務服務需要座席B處理，則座席A可以邀請座席B進行會議，一起進行處理。

Note :
會議電話時，必須保持電話為通話中才可以進行此操作。
A、單步會議
在電話輸入框中輸入需要參與會議的電話號碼或在浮動頁面中座席狀態查看中選擇需要參與會議的座席，點擊，此時可以實現三方通話。
B、多步會議
您可以在電話輸入框中輸入需要參與會議的電話號碼或在浮動頁面中座席狀態查看中選擇需要參與會議的座席，點擊，系統會對當前使用線路進行保持（Hold），並進行撥打加入會議的電話，再次點擊進行會議確認，此時可以實現三方通話。
Note :
在您做確認會議時，可以取消當前會議動作，點選會議邀請的線路，進行電話掛斷。此時會議邀請的線路為掛斷，主線路仍Hold，您如需取消Hold，可以再點擊Hold按鈕進行取消Hold。
系統支援多方會議，也就說，你可以在三方會議的情景下，再邀請第四方、第五方進行會議加入。

6、如何保持電話
保持電話，Hold，在座席與客戶進行通話服務時，可將電話進行保持，電話仍處於持續狀態，但客戶無法聽到座席的聲音，僅能聽到系統設置好的音樂音。

Note :
保持電話時，要求電話為通話中才可以進行此操作。
系統支援多線路同時Hold。
開始Hold：選擇需要Hold的線路，點擊 按鈕，系統對當前使用線路進行保持Hold。
取消Hold：選擇已Hold的線路，點擊 按鈕，系統對當前Hold線路取消保持，並恢復到通話中。

7、如何把電話轉到IVR
IVR，系統中的IVR流程設計，可以在AdminCenter中進行設計。座席在與客戶進行電話服務時，針對客戶的需求可將客戶的電話轉接到IVR中，由自動語音協助客戶完成電話服務。

Note :
要求電話為通話中才可以進行此操作。
系統支援兩種IVR轉接，一種是不回座席的IVR轉接，另一種是回座席的IVR轉接。
兩種的區別是：
① 不回座席的IVR轉接：電話直接轉接到IVR中，此電話不再轉回到座席，對於座席而言此電話服務已結束。
② 回座席的IVR轉接：電話轉接到IVR中，座席將繼續保持此電話，此電話服務並未結束。
